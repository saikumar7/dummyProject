Business Rules

Searching by Child's Name

  • When searching using “Child’s Name”, the report will pull a list of all children currently who have a custody start date with no custody end date.
    • Selecting a Child who has an address in MACWIS (on the demographics) will enable the Search option.
    • If no address for the child is entered in MACWIS,
      • the address will have to be entered in MACWIS first

        • OR
      • Select ‘Address’ to search by an address

Searching by Address

  • When searching using “Address”, the entire address (except Apt/Suite) is required as well as the Number of children in order for the search option to be available.

Search Results

  • Resource Homes are returned in the search results using the following criteria:
    • Within 50 miles (“as the crow flies”) of the child’s address displayed in the tool.
    • The Resource Home’s current license status is Active.
    • Provides one of the following three Placement Services: Foster Home, Adopter/Foster Adopt Home, Emergency Foster Home.
    • Has bed space available based on the age and gender of the child(ren).
  • Resource homes will NOT be included in the Initial Search Results if:
    • The distance between the Resource Home’s address is more than 50 miles from the child’s Address at Removal.
    • The home does not have space based on the age and gender of the child(ren).
    • There are 3 foster children currently placed in the home. (Regardless of the number of licensed beds in MACWIS).
    • There are 5 total children in the home (foster, biological, or adopted).
    • There are 2 children under the age of two in the home and the child needing placement is two years old or less.
  • Results indicate if there is a previous ANE on the home, regardless if the previous ANE was unsubstantiated. If there is a current ANE on the home, the home will not display in the results.
  • Advanced Search: There is an additional search option under the Advanced Search that will allow the distance between the Address at Removal and the Resource Home to be increased or decreased to find the best placement for the child.
    • NOTE 1: In order to clear the number of miles in the advanced search, it will be necessary to remove the miles entered (This will change in a future release).
    • NOTE 2: More Advanced Search options are coming in future releases.
